#MadSqlite Core

 * A simple C++14 [Sqlite](https://sqlite.org) abstraction
 * [FTS5](https://sqlite.org/fts5.html) and [RTree](https://www.sqlite.org/rtree.html) extension modules enabled
 * [BSD License](LICENSE.md)
